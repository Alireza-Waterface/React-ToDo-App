To-Do app built using React.js

Features:
	1. Dark/Light mode
	2. Filter tasks
		a) All
		b) Completed
		c) Incomplete
	3. Sort tasks by priority
	4. Editing task title
	5. Deleting tasks
	6. Notes/Description for task
	7. Save tasks to browser local storage

 App available on: https://todo-wf.netlify.app
